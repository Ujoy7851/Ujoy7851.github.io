TOEIC 시험대비 정리

Mar 25, 2019 in Toeic

본 포스팅은 YBM 전략 토익 RC을 보고 정리한 글입니다.

# UNIT 1 문장구조

주어

  • 명사, 대명사, 동명사구, 명사절

동사

  • 주어 뒤에 위치
  • 동사, 조동사+동사원형 O
  • to부정사, 동명사, 분사 X
  • 자동사 + (보어)
  • 타동사 + 목적어 + (목적어, 목적보어)

목적어

  • 타동사 뒤, 전치사 뒤에 위치
  • 명사, 대명사, 동명사구, to부정사구, 명사절

보어

  • 2형식 또는 5형식
  • 형용사, 명사, to부정사, 분사, 동사원형

가주어 it: 주어가 to부정사구, that절 같이 긴 경우 주어 자리에 it을 대신 씀
가목적어 it: 5형식 동사 뒤 it은 가목적어로, 뒤에 진짜 목적어(to부정사, that절)이 옴

1. 주어나 목적어 자리에는 명사

  • 주어 자리: 문장 맨 앞 또는 동사 앞쪽
  • 목적어 자리: 타동사나 전치사 뒤
  • 형용사/소유격/접속사 + 명사

2. There is 뒤에는 명사

  • There + be동사에서는 be동사 뒤가 주어 자리
  • is + 단수명사
  • are + 복수명사
  • There remain, There exist도 뒤의 명사에 수 일치

3. 절이 한 개면 동사도 한 개

  • 절에는 반드시 하나의 동사 필요
  • 동사 O: 동사, 조동사+동사원형, be동사+ing/p.p, have+p.p
  • 동사 X: to+동사원형, 분사(ing/p.p), 동명사

4. 빈칸 뒤에 목적어 없으면 자동사 / 목적어 있으면 타동사

  • 자동사 + 부사, 전치사
  • 타동사 + 목적어
자동사 타동사
rise, fall, arrive, function, work, emerge, commence, proceed, expire, occur, happen, take place appoint(임명하다, 지정하다), enter, attend, announce, show, award, leave, keep, find, honor, consider, deem, make, acquire

5. be동사 뒤에는 형용사

  • be동사/2형식동사 + 보어
  • 보어: 명사, 형용사

6. keep + 명사 뒤에는 형용사

  • 5형식 문장의 목적격보어: 명사, 형용사
2형식 동사 5형식 동사
be, become, turn, remain, stay, feel, prove, look, seem, appear keep, fine, consider, deem, make, leave

7. part6 - 빈칸 뒤 문장의 대명사는 주로 앞 문장의 명사를 대신하는 말

# UNIT 2 명사

형태

주로 tion, sion, ment, ance, ness, ty로 끝남

역할

주어, 목적어, 보어

명사의 수

명사는 셀 수 있는 명사와 셀 수 없는 명사로 구분

가산 단수명사 가산 복수명사 불가산 명사
앞에 반드시 한정사가 옴 뒤에 (e)s가 붙음 단수취급, 복수형태X
customer price, reeport, rate, fee, expense, cost, fund, discount, order, load, estimate, account, permit, certificate, document, task, approach information, equipment, merchandise, access, baggage, luggage, approval, advice, knowledge, consent, research, work, merchandise, interest, permission, machinery, furniture, money, cash

한정사

명사의 종류에 따른 한정사의 종류가 정해짐

구분 종류 단수 앞 복수 앞 불가산 앞
관사 a, an o
the o o o
소유격 my, its, Joy’s o o o
지시형용사 this, that o o
these, those o
부정형용사 any, no o o o
all, most, more, some, other o o
many, few, both, several o
much, little o
each, every, another, either o

8. 주목보는 명사 자리

9. 관형명 소형명

  • 관사, 소유격 뒤에는 반드시 명사가 와야 함
  • 사이에 주로 형용사가 붙음
  • 관부형명도 가능

10. estimate는 명사도 됨

  • 명사, 동사의 형태가 같은 어휘: advance, supply, recruit(신입사원,채용하다), project, support, function, delay, use, study, limit, charge, demand, benefit, request, purchase, monitor, access, approach, proceed(진행, 진행하다), proceeds(수익금)
  • 명사, 형용사의 형태가 같은 어휘: official, individual, potential, editorial, disposable(일회용품, 처분할 수 있는), executive, alternative, specific, objectvie, representative, professional, characteristic

11. 복합명사

  • 복합명사: 두 개의 명사가 모여 하나의 명사처럼 쓰임
  • 그냥 외워라

12. 주어와 동사 수일치

  • 단수, 불가산명사 + 단수동사
  • 복수명사 + 복수동사
  • 수식어가 복수여도 상관없이 동사에 수 일치 (Our reputation is ~)

13. 불가산명사는 뒤에 (e)s 붙지 않음

  • 불가산명사는 셀 수 없으므로 단수 형태
  • all, many + 복수명사
  • much, little + 불가산명사
  • 위에 명사의 수 참고

14. 빈칸 앞에 관사, 소유격이 없으면 가산 단수명사 X

  • 가산 단수명사는 반드시 한정사 필요
  • 한정사 없을 경우 복수형태로
  • 불가산명사 위에 명사의 수 참고

15. 보기에 명사가 여러개 -> 문맥에 맞는 걸 골라라

ex. resident 거주자 / residence 거주, 주택
correspondent 기자 / correspondence 서신

# UNIT 3 대명사

인칭대명사

i, my, me, mine, myself….

  • 주격, 소유격, 목적격

  • 재귀대명사

    • 재귀용법: 목적어로 쓰임, 주어와 목적어가 같을 때
    • 강조용법: 명사 강조, 강조하는 말 바로 뒤나 문장 끝에

  • 소유대명사: 소유격+명사를 대신하는 말.

지시대명사

  • 단수: this, that
  • 복수: these, those
  • that, those: 문장 내에서 앞에 나온 명사 대신함. 일반적으로 뒤에 수식어 옴
  • 형용사 역할: 명사 앞에서 지시형용사로 쓰임
    • this/that + 단수명사
    • these/those + 복수명사

부정대명사

하나, 일부, 대부분같이 분명하게 정해지지 않은 대상을 지칭

구분 부정대명사
단수 취급 one, each, either, another
복수 취급 (a) few, both, several, many
불가산(단수 취급) (a) little, much
복수 또는 불가산 all, most, half, any, some

17. 동사 앞은 주격, 동사/전치사 뒤는 목적격

18. 명사 앞은 소유격

  • 소형명
  • 소유격 + own + 명사

19. 주어와 목적어 같으면 목적어로 재귀대명사

  • 명령문은 주어 you가 생략되므로 재귀대명사로 yourself를 씀

20. 주어 뒤, 목적어 뒤는 재귀대명사

  • 명사 강조. 직접, 몸소라는 뜻

21. by 뒤에는 재귀대명사

  • by oneself: 혼자서
  • one ones’ own: 혼자서
  • for oneself: 스스로, 자신을 위해
  • in itself: 그 자체가, 본질적으로

22. those + who/p.p/전치사

  • 관계사절이나, 분사, 전치사구의 수식을 받아 ‘~한 사람들’이라는 뜻으로 쓰임
  • they, them, we, this, that 등은 이런 용도로 쓰이지 않음
  • anyone who + 단수동사: ~하는 누구라도

23. 부정대명사 자리라면 수 파악

  • 어느 명사를 대신하는지 파악하여 수 일치
  • every, other은 형용사

24. 부정대명사 표현

  • one - another
  • some - others
  • 관사 + 형용사 + one: ~한 것
  • each other, one another은 주어가 될 수 없음

25. 접속부사

앞뒤 문장이 어떤 관계로 연결되었는지 파악하는 단서

구분 접속부사
인과 therefore, thus, hence, consequently, as a result
역접 however, even so, in contrast, to the contrary, nevetheless, nonetheless
시간 then, afterwards, subsequently, finally
추가 furthermore, moreover, besides, plus, also, likewise, additionally, in addition
조건 if so, if not, otherwise, in this case, accordingly
기타 in fact, alternatively, instead, unfortunately, regrettably, after all, for example, for instance

# UNIT 4 동사

수일치

구분 3인칭 단수 복수, 2인칭 단수
be동사 is, was are, were
do does do
have has have
일반 -(e)s 동사원형
  • 수식어는 수일치와 무관
  • 관계대명사절의 동사는 관계대명사 앞 명사에 수일치

수량 표현의 수일치

전부나 일부를 뜻하는 부정대명사일 경우 주의!

  • all/most/some/any + of the + 복수명사 + 복수동사
  • all/most/some/any + of the + 불가산명사 + 단수동사
    ex. All of the programs are listed on the web site.
    ex. All of the information is listed on the web site.
  • one, each, either, neither + of the + 복수명사 + 단수동사

능동태, 수동태

  • 능동태: 주어 + 타동사 + 목적어
  • 수동태: 주어 + be p.p (+ by + 행위자)

시제

구분 현재 과거 미래
단순 am, are, is, 동사원형(es) was, were, 동사원형ed will + 동사원형
진행 am/are/is Ving was/were Ving will be Ving
완료 have/has p.p had p.p will have p.p

  • 타동사 뒤에 목적어 없으면 수동태

29. 수동태 뒤에 목적어 있는 동사

  • 4형식 동사는 수동태가 되도 목적어 필요
  • 5형식 동사는 수동태 되면 뒤에 목적보어(명사, 형용사)

    be made ~
    be deemed ~
    be considered ~
    be held ~ (~라고 생각하다)
    be named ~

30. 명령문이나 조동사 뒤에 동사원형

  • If절/When절 + 명령문: ~하면/할 때 ~해라
  • 조동사 + 부사(not, no longer, finally, probably) + 동사원형

31. yesterday 있으면 과거시제

  • 과거시제: yesterday, ago, last, in 1999, previously, formerly, originally, initially, recently(현재완료도 가능)
  • 현재시제: 현재의 상태, 일반적 사실, 반복적인 일을 나타냄. now, currently, every, normally, 빈도부사(usually, always, often)

32. over the past ~ 있으면 have p.p

  • 현재완료: 과거부터 현재까지 계속되거나 현재에 완료된 상태
  • over/for/during/in the past/last + 기간
  • since + 과거

33. next 있으면 will

  • next, tomorrow, soon, until further notice, as of(~부터)
  • 현재진행이 가까운 미래의 일정을 나타내기도 함

34. If + 주어 + 현재시제, 주어 + will

  • If/When + 주어 + 현재시제, 주어 + will
  • 시간 접속사: when, befor, after, while, until, as soon as
  • 조건 접속사: if, unless, in case (that), providing(that), provided (that)

35. by the time + 주어 + 과거시제, 주어 + had p.p

  • 과거완료: 과거의 사건보다 이전에 일어난 일
    • by the time / before + 주어 + 과거시제, 주어 + had p.p
    • after 주어 + had p.p, 주어 + 과거시제
  • 미래완료
    • by the time 주어 + 현재시제, 주어 + will have p.p
